
# array-last

  Return the last elements of an Array

## Installation

  Install with [component(1)](http://component.io):

    $ component install leafs/array-last

## Usage

```js
var last = require('last');
last(['olivier', 'bruno', 'amy']);
// ['amy']

last(['olivier', 'bruno', 'amy'], 2);
//['bruno', 'amy']
```
